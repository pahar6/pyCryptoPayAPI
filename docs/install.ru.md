# Установка
Библиотеку можно установить несколькими способами:
=== "Через PYPI"
    ```sh
    pip install pycryptopay-sdk
    ```

=== "Через pip+git"
    ```sh
    pip install git+https://github.com/LulzLoL231/pyCryptoPayAPI.git
    ```

=== "Из исходников"
    ```sh
    pip install . # (1)!
    ```

    1. Выполнять в папке с библиотекой
