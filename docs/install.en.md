# Install
The library can be installed in several ways:
=== "With PYPI"
    ```sh
    pip install pycryptopay-sdk
    ```

=== "With pip+git"
    ```sh
    pip install git+https://github.com/LulzLoL231/pyCryptoPayAPI.git
    ```

=== "From source"
    ```sh
    pip install . # (1)!
    ```

    1. Execute in root folder.
